Mundial 2026 - versión web del Excel

Cómo usar:
1. Descomprime este ZIP.
2. Abre index.html en Chrome, Edge o Firefox.
3. Edita celdas directamente.
4. Usa "Guardar cambios" para guardar en el navegador.
5. Usa "Exportar hoja a CSV" si quieres llevar una hoja de vuelta a Excel.

Nota: Es una página estática; no necesita servidor ni internet.
